## Quick Setup

### Local Development
```bash
php -S localhost:8000 -t site/
```
Then open `http://localhost:8000` in browser.

### Deployment
Requires PHP-enabled hosting (cPanel, cloud platforms, VPS with LAMP/LEMP stack).
**Note**: Static hosting (Netlify, GitHub Pages) will NOT work.
